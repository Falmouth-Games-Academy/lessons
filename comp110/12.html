This week looks at how the source code of a program is translated into a form that the computer can execute. We introduce machine code, the low-level format of a program as executed by the CPU.

Learning Objectives:
  • Distinguish between ahead-of-time compilation, just-in-time compilation and interpretation
  • Describe the basic operation of an assembler, compiler and interpreter
  • Write and understand simple assembly language programs
