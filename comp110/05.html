This week introduces computability and complexity. Writing efficient programs often starts with choosing the correct algorithms; this session will equip you to reason about which algorithm is appropriate for a given situation.

Learning Objectives:
  • Explain the notion of computational complexity
  • Use "big O" notation to express computational complexity
  • Apply appropriate algorithms to achieve efficiency
